Slider class constructor.

Parameters
maxStateMaximum slider state. Minimum state is 0.
initStateSlider state upon creation. If bigger than maxState, it's set to floor(maxState/2) and an error is written to the log.
positionPosition of the slider
wholeSizeSize of the whole Slider represented as vec2(width, height)
buttonSizeSize of the left button represented as vec2(width, height). Right button will have the same size.
sliderSizeSize of the actual sliding part represented as vec2(width, height)
texBGBackground Texture file name
texSliderTexture file name for the sliding part
texLeftUnpressedTexture file name for left-side button when unpressed. Right-side button is rendered as a vertical flip of the left one.
texLeftPressedTexture file name for left-side button when pressed. Right-side button is rendered as a vertical flip of the left one.
